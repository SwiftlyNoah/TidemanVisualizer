About This Project

The Tideman Visualizer was created to demonstrate how ranked-choice (Tideman) voting works and to bridge the gap between algorithmic logic and intuitive understanding. It draws directly from CS50â€™s problem set on the Tideman algorithm and presents both C and JavaScript implementations side by side.

Credits & Acknowledgments

The problem definition and the original C starter code come from CS50. The C implementation used in this project is identical to the version I submitted for CS50 Problem Set 3. The simulator and web visualizer were built later in JavaScript, following the exact same logic and control flow.

You can view the full project source code on GitHub: github.com/SwiftlyNoah/TidemanVisualizer. Contributions, suggestions, and improvements are always welcome!
