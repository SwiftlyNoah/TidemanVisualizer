Tideman Election Visualizer

An interactive demo of the Tideman ranked-choice voting algorithm, and a walkthrough of its implementation in C and JavaScript.

Try the Simulator See How It Works Dig Into the Code

1. Collect Rankings

Voters rank candidates in order of preference. The algorithm builds a preference matrix counting how many voters prefer each candidate over another.

2. Create Winner–Loser Pairs

Each pair of candidates is compared head-to-head. The stronger victories are recorded and sorted by their margin.

3. Lock Pairs & Find the Winner

Pairs are locked into a directed graph unless they form a cycle. The graph’s source (no incoming edges) is declared the Tideman winner.
