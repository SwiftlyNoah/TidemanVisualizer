Interactive Tideman Simulator

Enter candidate names and voter rankings. JavaScript will replicate the Tideman algorithm: build preferences, sort pairs, lock edges, and display the winner (or report that there is no clear winner).

1. Candidates

Enter up to four candidates. Leave unused fields blank.

2. Voter Ballots

For each voter, rank all candidates. Each candidate must appear exactly once in each ballot.

3. Results
